Tags, anidamiento, estructura de una página HTML

  • Una página HTML está formada por tags. Cada tag tiene un identificador (o nombre).

    P.ej. para marcar un título del nivel más alto se usa el tag h1

  • Hay que abrir y cerrar cada tag.

    Para abrir un tag, se pone el identificador entre corchetes triangulares, p.ej.

    <h1>

    Para cerrar un tag, lo mismo, anteponiendo una barra al identificador del tag. P.ej.

    </h1>

  • Entre la apertura y el cierre del tag, se pone el contenido, que puede ser

    • texto que se escribe derecho viejo, este texto va a formar parte de la página
    • uno o varios tags, en el caso de tags anidados
    • una mezcla

    P.ej. este código HTML especifica un título de nivel principal

    <h1>Estructura general de una célula</h1>

    No es necesario que vaya todo en una misma línea, ni que no haya espacios entre tag y contenido. Lo anterior es equivalente a

    <h1>

      Estructura general de una célula

    </h1>

    y también a

    <h1> Estructura general de una célula </h1>

  • Los tags están anidados. En este ejemplo

    <body>

      <h1>Este es el título</h1>

    </body>

    el tag h1 se está usando anidado dentro del tag body .

  • La estructura de cualquier página HTML es

    <html>

      <head>

        ... acá van indicaciones generales para la página ...

      </head>

      <body>

        ... acá va el contenido de la página ...

      </body>

    </html>

  • El catálogo de todos los tags que se pueden usar en una página HTML está en https://www.w3schools.com/tags/default.asp .

    Los que son solamente para HTML5 no sé si conviene usarlos, tal vez no anden en máquinas viejas.

Atributos, estilos, CSS

  • Todo tag dentro de una página puede (o no) tener atributos. Cada atributo tiene un identificador (o nombre), y un valor.

    P.ej. en el tag img , que sirve para insertar una imagen en la página, el archivo con la imagen se especifica en el atributo src. P.ej.

    <img src="images/kokito-beach.jpeg"></img>

    Acá al tag img se le está poniendo un atributo, con identificador src y valor images/kokito-beach.jpeg. Por lo general, el valor de los atributos se pone entre comillas dobles.

    Dicho de otra forma, al tag img le pusimos images/kokito-beach.jpeg como valor del atributo src.

  • De paso, lo anterior es equivalente a

    <img src="images/kokito-beach.jpeg"/>

    O sea: si un tag no tiene contenido, entonces se puede abrir y cerrar de una.

  • Algunos atributos tienen sentido solamente para un tag, o un conjunto reducido de tags. P.ej. el atributo src se usa solamente en el tag img (entre los tags que yo conozco, puede ser que se use en algún otro tag).

    Otros atributos son universales, o sea, tienen sentido en cualquier tag. El ejemplo más importante es style.

  • Con el atributo style se le pueden dar un montón de indicaciones de estilo a un tag HTML.

    P.ej. para poner un título en rojo se pone

    <h1 style="color: red;">Estructura general de una célula</h1>

    En el valor del atributo style se indica una o varias propiedades. Cada propiedad tiene un nombre y un valor. En el ejemplo de recién, al tag h1 le ponemos un estilo que tiene una sola propiedad, de nombre color y valor red. Cada propiedad se termina con un punto y coma.

  • Un ejemplo con dos propiedades. Para poner un título en color rojo y fuente Arial, ponemos

    <h1 style="color: red; font-family: Arial">Estructura general de una célula</h1>

    Acá las propiedades son color y font-family, con valores red y Arial respectivamente.

  • El lenguaje CSS especifica todas las propiedades que podemos poner dentro de un style

    Hay un catálogo de propiedades en https://www.w3schools.com/cssref/default.asp .

El tag span

¿Qué pasa si queremos que una mitad de un renglón aparezca en rojo y la otra en verde.? O sea, que aparezca algo así

    Hola amigos y compañeros

Los estilos solamente pueden usarse en un tag. Entonces, este renglón va a necesitar dos tags, uno para la parte roja, otro para la verde.

Para estos casos se inventó el tag span. Este tag no hace nada, solamente sirve para poder marcar un estilo a una parte del texto.

Lo de arriba se resuelve así:

<p>

  <span style="color=red">Hola amigos </span>

  <span style="color=green">y compañeros</span>

</p>
